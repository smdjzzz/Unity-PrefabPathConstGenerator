# PrefabPathConstGenerator

A Unity Editor window tool that scans `.prefab` files from a specified folder and generates a C# static class containing path/name constants — so you never have to write or mistype prefab path strings by hand.

## Features

- **Folder scanning** — point at any folder in the Project window (supports drag-and-drop).
- **Recursive scan** — optional toggle to include all sub-folders.
- **Cherry-pick prefabs** — a checkbox list lets you include/exclude individual prefabs before generating.
- **Two path formats**:
  - Full relative path, e.g. `"Prefabs/UI/MainPanel"`
  - Name only, e.g. `"MainPanel"`
- **Custom class name** — name the generated static class however you like.
- **Smart naming** — constants use `UPPER_SNAKE_CASE`, duplicate names get auto-incremented suffixes, and the `=` signs are aligned for readability.
- **Persistent settings** — all your input/output folder paths, class name, and toggles are saved to a JSON file next to the script. No registry, no EditorPrefs. Reopen the window and everything is exactly where you left it.
- **Auto-refresh** — calls `AssetDatabase.Refresh()` after generation so the new script is immediately available.

## Installation

Copy the `Editor` folder into your Unity project under any `Assets/**/Editor` path:

```
YourProject/
└── Assets/
    └── (any path)/
        └── Editor/
            ├── PrefabPathConstGenerator.cs
            └── PrefabPathConstGenerator.cs.meta
```

Unity will automatically detect it as an Editor script.

## Usage

1. Open the tool via **Tools > 预制体路径常量生成器** in the Unity menu bar.
2. Drag a folder into the **Target Folder** field — this is where your `.prefab` files live.
3. Drag a folder into the **Output Folder** field — this is where the generated `.cs` will be written.
4. Optionally change the **Class Name**, toggle **Include Sub-folders**, and pick a path format.
5. Click **扫描预制体** to scan.
6. Check/uncheck individual prefabs in the list.
7. Click **生成常量脚本**.

### Generated output example

```csharp
// <auto-generated>
//   由 预制体路径常量生成器 自动生成
//   生成时间: 2026-04-21 20:30:00
//   请勿手动修改此文件，修改将在下次生成时被覆盖
// </auto-generated>

public static class PrefabPath
{
    public const string ENEMY_SKELETON      = "Prefabs/Enemies/Skeleton";
    public const string ENEMY_SKELETON_2    = "Prefabs/Enemies/Skeleton_Archer";
    public const string UI_MAIN_PANEL       = "Prefabs/UI/MainPanel";
    public const string UI_SETTINGS_PANEL   = "Prefabs/UI/SettingsPanel";
}
```

Then use it in code:

```csharp
var panel = Instantiate(Resources.Load<GameObject>(PrefabPath.UI_MAIN_PANEL));
```

## Requirements

- Unity 2021.3+ (uses standard `EditorWindow` / `EditorGUILayout` APIs)
- No external dependencies
