# PrefabPathConstGenerator

一个 Unity 编辑器窗口工具，扫描指定文件夹中的 `.prefab` 文件，自动生成包含路径常量的 C# 静态类——告别手写字符串和拼写错误。

## 功能特性

- **文件夹扫描** — 在 Project 窗口中拖入任意文件夹即可指定扫描目标。
- **递归扫描** — 可选开关，勾选后自动扫描所有子文件夹。
- **逐个勾选** — 扫描结果以勾选列表展示，生成前可自由选择包含/排除哪些预制体。
- **两种路径格式**：
  - 完整相对路径，如 `"Prefabs/UI/MainPanel"`
  - 仅名称，如 `"MainPanel"`
- **自定义类名** — 生成的静态类名称完全由你决定。
- **智能命名** — 常量自动转为 `UPPER_SNAKE_CASE`，同名冲突自动追加数字后缀，等号自动对齐。
- **设置持久化** — 输入/输出路径、类名、所有开关均保存为脚本同目录下的 JSON 文件。不写注册表，不使用 EditorPrefs。关闭窗口后再次打开，所有设置原样恢复。
- **自动刷新** — 生成后自动调用 `AssetDatabase.Refresh()`，新脚本立即可用。

## 安装

将 `Editor` 文件夹复制到你的 Unity 项目中任意 `Assets/**/Editor` 路径下即可：

```
YourProject/
└── Assets/
    └── (任意路径)/
        └── Editor/
            ├── PrefabPathConstGenerator.cs
            └── PrefabPathConstGenerator.cs.meta
```

Unity 会自动将其识别为编辑器脚本。

## 使用方法

1. 在 Unity 菜单栏中点击 **Tools > 预制体路径常量生成器** 打开窗口。
2. 将一个文件夹拖入 **目标文件夹** 字段——这是你的 `.prefab` 文件所在目录。
3. 将一个文件夹拖入 **输出文件夹** 字段——生成的 `.cs` 文件将保存在此处。
4. 按需修改 **类名**、切换 **包含子文件夹**、选择路径格式。
5. 点击 **扫描预制体**。
6. 在列表中勾选/取消勾选需要包含的预制体。
7. 点击 **生成常量脚本**。

### 生成示例

```csharp
// <auto-generated>
//   由 预制体路径常量生成器 自动生成
//   生成时间: 2026-04-21 20:30:00
//   请勿手动修改此文件，修改将在下次生成时被覆盖
// </auto-generated>

public static class PrefabPath
{
    public const string ENEMY_SKELETON      = "Prefabs/Enemies/Skeleton";
    public const string ENEMY_SKELETON_2    = "Prefabs/Enemies/Skeleton_Archer";
    public const string UI_MAIN_PANEL       = "Prefabs/UI/MainPanel";
    public const string UI_SETTINGS_PANEL   = "Prefabs/UI/SettingsPanel";
}
```

在代码中使用：

```csharp
var panel = Instantiate(Resources.Load<GameObject>(PrefabPath.UI_MAIN_PANEL));
```
